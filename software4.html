STEGANOGRAPHY

The user-oriented program for my personal portfolio is designed to provide a seamless experience for users to perform steganographic encoding and decoding tasks. The program prompts the user to select their desired function, whether it's encoding or decoding. It also requests the user to choose an image for the steganographic process. With a user-friendly interface, the program enables text entry for encoding, allowing users to input the desired hidden message. This project showcases my ability to create intuitive software that prioritizes user experience while implementing advanced techniques like steganography.
