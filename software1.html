QUIZ GAME

The Console-based Quiz Game Software, developed in the C language, allows users to input answers to multiple-choice questions. Users have the flexibility to change the question category. Points are awarded for correct attempts, and the final accumulated points are displayed at the end. Key features include user input handling, category selection functionality, scoring system with point rewards for correct answers, and a console-based user interface. The software can be implemented using C language constructs, such as loops, conditionals, and arrays. The user's progress, points, and final score can be stored in variables. The software can be compiled and executed directly in a console environment.
